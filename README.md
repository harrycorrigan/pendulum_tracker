# Pendulum Tracker
Project for tracking a pendulum in a very particular set of videos with no flexibility and analyzing the resulting data. 
Sample data is included but not original videos.

# Usage
Clone Repo  
```
git clone https://github.com/harrycorrigan/pendulum_tracker
```

CD into directory  
```
cd pendulum_tracker
```

Install using poetry  
```
pip install poetry  
poetry install
```

Run the program  
```
python pendulum_tracker.py
```
